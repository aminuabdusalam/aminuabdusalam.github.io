Team project (5 devs). We wanted to eliminate the "is this a carry-on?" anxiety at the gate. My contribution was the backend — I built the Django REST API and was solely responsible for the OpenCV bag-scanner service, which uses a top-view and side-view phone photo plus a reference object to estimate real-world luggage dimensions and classify the bag as carry-on vs. checked. Teammates built the React Native front-end that talked to my API.
